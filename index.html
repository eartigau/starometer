STAR-O-METER

Enter a stellar identifier (SIMBAD-resolvable name or alias). The tool cross-matches astrometry and broadband photometry from Gaia DR3, 2MASS, and AllWISE (0.5–22 μm). For M dwarfs it applies the existing empirical calibrations; for earlier main-sequence stars it uses Gaia DR3 astrophysical parameters explicitly labeled in the output. the spectral energy distribution and Hertzsprung–Russell diagram position relative to the solar-neighbourhood sample (<20 pc) are displayed below.

Requires a same-origin (or CORS-permitting) HTTP server; opening as a plain file:// URL will block cross-origin TAP queries.
